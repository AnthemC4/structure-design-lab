# 吸管搭建实验室 - 使用说明

## 📱 使用方式

### 方式 1：本地打开（推荐）
1. 双击 `吸管搭建实验室.html` 文件
2. 在浏览器中直接使用
3. 支持电脑/手机/平板

### 方式 2：GitHub Pages 部署（在线分享）
1. 登录 GitHub
2. 创建新仓库（如 `straw-lab`）
3. 上传此 HTML 文件
4. 启用 GitHub Pages
5. 获得永久在线链接

## 🎮 操作指南

### 添加吸管
- 点击「添加吸管」→ 选择方向（水平/垂直/斜向 45°）
- 最多 20 根吸管

### 编辑吸管
- 点击吸管选中（变红）
- 使用「左旋/右旋」调整角度
- 拖拽移动位置（自动吸附连接点）

### 删除吸管
- 点击「删除模式」→ 点击吸管删除

### 保存设计
- 5 个设计槽位（方案 1-5）
- 点击「保存设计」存入当前槽位
- 点击槽位可加载已存设计

### 撤销操作
- 点击「撤销」回退上一步

## 📋 教学建议

**适用课程：** 教科版科学 四年级/六年级 结构与设计单元

**探究目标：**
- 理解三角形稳定性
- 探索结构承重原理
- 设计优化思维

**课堂活动：**
1. 基础搭建：完成指定形状
2. 挑战任务：用最少吸管搭建最高塔
3. 创意设计：自由创作结构
